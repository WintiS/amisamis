# amis amis - Construction Company Website

One-page, no-scroll React website for the construction company "amis amis". The site displays an "Under Construction" message in Czech with contact information.

## Technologies

- React 18
- Vite
- Tailwind CSS

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Features

- Single page layout (100vh, no scroll)
- Under construction message in Czech
- Contact information section (email, phone, address)
- Image placeholder ready for future image
- Construction-themed styling with safety colors
